Citi Bike E-Bike Performance Hub

Case study com dados reais do Citi Bike para analisar performance operacional, adoção de e-bikes, pressão por estação e grupos comportamentais de viagem com OLTP, OLAP, dbt, Streamlit e Machine Learning.

O que este projeto responde

  • que estações ficam mais perto de rutura ou saturação;
  • como varia o uso de e-bikes ao longo do dia;
  • que grupos de viagem existem no sistema;
  • como montar uma stack analítica com dados reais públicos.

Quick Start

python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python python/run_pipeline.py --months 1 --sample-rows-per-file 20000
python python/ml_citibike.py
streamlit run streamlit/app.py

Porque este caso é forte para portefólio

Problema de negócio

Foca risco operacional, e-bike share e pressão sobre estações com perguntas que fazem sentido no mundo real.

Camada analítica

Cria fact tables e dimensões em DuckDB para responder rapidamente a perguntas de performance e procura.

Machine Learning

Inclui classificação de uso de e-bike e clustering de perfis de viagem, com limitações dos dados explicadas.
